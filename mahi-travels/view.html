How to View Mahi Travels Website

Method 1: Direct File Opening

  1. Open File Explorer
  2. Navigate to: C:\Users\som\mahi-travels
  3. Double-click on "index.html"
  4. The website should open in your default web browser

Method 2: Using Web Browser

  1. Open any web browser (Chrome, Firefox, Edge, etc.)
  2. Press Ctrl + O (Open file)
  3. Navigate to: C:\Users\som\mahi-travels
  4. Select "index.html" and click Open
